# The Odin Project: Basic Ruby Projects

Section for the basic Ruby exercises from The Odin Project's Ruby course.
See the repository index at [../README.md](../README.md).

This directory collects exercises from the basic Ruby part of the course. Each subfolder contains an exercise and its tests.

## Projects

- [cipher](cipher/) — Caesar cipher implementation.
- [substrings](substrings/) — Substrings counter exercise.
- [stock](stock/) — Stock picker exercise.
- [bubble_sort](bubble_sort/) — Bubble sort exercise and tests.

---

## Running tests

Run RSpec for a specific project by changing into its folder and running the test command. Example:

```bash
cd substrings
rspec --format documentation spec
```
