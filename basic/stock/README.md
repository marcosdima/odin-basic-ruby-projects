# ruby-stock-picker
Exercise from https://www.theodinproject.com/lessons/ruby-stock-picker

# Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/marcosdima/odin-ruby-projects.git
   ```

2. **Navigate to the folder:**
   ```bash
   cd odin-ruby-projects/basic/stock
   ```

3. **Run the script:**
   ```bash
   ruby stock_picker.rb "1,5,3,5"
   ```

   - The script accepts a comma-separated list of prices as the first argument. If no argument is provided, a default sample array is used.

4. **Example output:**
   ```bash
   ruby stock_picker.rb
   ```
   Output: `[0, 1]`

   ---

   ```bash
   ruby stock_picker.rb "13,5,3,5,10"
   ```
   Output: `[2, 4]`
