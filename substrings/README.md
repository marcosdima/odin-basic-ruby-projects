# ruby-substrings
Exercise from https://www.theodinproject.com/lessons/ruby-sub-strings

# Usage

1. **Clone the repository:**
	```bash
	git clone https://github.com/marcosdima/odin-basic-ruby-projects.git
	```

2. **Navigate to the folder:**
	```bash
	cd odin-basic-ruby-projects/substrings
	```

3. **Run the script:**
	```bash
	ruby substrings.rb DICTIONARY WORD
	```

4. **Examples:**
	```bash
	ruby substrings.rb
	```
	Output: `{"how" => 2, "howdy" => 1, "part" => 1, "partner" => 1, "it" => 2, "i" => 3, "sit" => 1, "down" => 1, "own" => 1, "go" => 1, "going" => 1}`
---
    ```bash
	ruby substrings.rb "a,b,c,!" Hola!
	```
	Output: `{"a" => 1, "!" => 1}`
